# v0.0.0.1 リリースノート

リリース日: 2026-06-15

## 概要

このリリースは、公開版 ISLES 2022 2D+3D blend pipeline リポジトリに対する最初の GitHub Release を文書面まで含めて正式化するものです。コードベース自体は引き続き「データなし・重みなし」の公開バンドルですが、今回のリリースでは来歴、再現性の範囲、公開物の境界を説明するための文書を揃えています。

## このリリースに含まれるもの

- 2D + 3D + 2.5D の異種アンサンブル推論パイプラインの公開ソース一式
- Apple Silicon 向け MPS 互換 nnU-Net 3D trainer workaround
- データ不要のリポジトリ検証用 smoke test
- audit map、citation metadata、sample manifest、英日エントリーガイド
- 実験ジャーニー文書の英語版・日本語版
- v0.0.0.1 向け GitHub Release ノートと changelog

## リポジトリ結果スナップショット

- ISLES 2022 の 25 ケース独自 hold-out split mean Dice: 0.7527
- 最終 blend: 0.20 ConvNeXt + 0.80 nnU-Net blend
- nnU-Net blend: 0.60 2D 3-fold 平均 + 0.40 3D 2-fold 平均
- 適応閾値: base 0.30、予測ボリュームが 4000 voxel を超える場合は 0.03 に切替

## v0.0.0.0 からの変更点

- 実験ジャーニー文書の英語版・日本語版を追加
- GitHub Release 用の英語版・日本語版リリースノートを追加
- トップレベルの changelog を追加
- 公開メタデータの version 表記を v0.0.0.1 に統一

## 公開アーティファクトの境界

含まれるもの:
- 手法確認に必要なソースコードと設定
- 公開バンドルを監査するためのドキュメント

含まれないもの:
- 医療データセット
- 学習済みモデル重み
- 非公開の実験出力、ログ、run ディレクトリ

## 検証

