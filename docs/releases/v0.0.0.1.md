# v0.0.0.1 Release Notes

Release date: 2026-06-15

## Summary

This release formalizes the first documented GitHub Release for the public ISLES 2022 2D+3D blend pipeline repository. The codebase remains intentionally data-free and weight-free, while the release package now includes the documents needed to explain provenance, reproducibility scope, and the exact public artifact boundary.

## Included in this release

- Public source bundle for the heterogeneous 2D + 3D + 2.5D ensemble pipeline.
- MPS-compatible nnU-Net 3D trainer workaround for Apple Silicon.
- No-data smoke test for quick repository verification.
- Audit map, citation metadata, sample manifest, and bilingual repository entry guides.
- Scientific experiment journey documents in English and Japanese.
- Dedicated GitHub Release notes and changelog entries for v0.0.0.1.

## Repository result snapshot

- 25-case custom ISLES 2022 hold-out split mean Dice: 0.7527
- Final blend: 0.20 ConvNeXt + 0.80 nnU-Net blend
- nnU-Net blend: 0.60 2D 3-fold average + 0.40 3D 2-fold average
- Adaptive threshold: base 0.30, fallback 0.03 when predicted volume exceeds 4000 voxels

## Changes since v0.0.0.0

- Added English and Japanese scientific experiment journey documents.
- Added English and Japanese release notes for the GitHub Release.
- Added a top-level changelog.
- Aligned public metadata version fields with v0.0.0.1.

## Artifact boundary

Included:
- Repository source code and configs required to inspect the method
- Documentation needed to audit the public bundle

Excluded:
- Medical datasets
- Trained model weights
- Private experiment outputs, logs, and run directories
